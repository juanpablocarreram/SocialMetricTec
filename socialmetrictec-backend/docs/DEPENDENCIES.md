### Database Setup Instructions
This document outlines the procedure for installing all dependencies and make the API listen to requests

1. Make sure to read the DATABASE.md file and complete the database config with the initial admin user
2. Create a python venv and once inside that venv, install all the dependencies listed in requirements.txt with the following command:
    pip3 install -r requirements.txt
3. Run the server wit the following command:
    fastapi dev
